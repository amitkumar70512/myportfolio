# Images Directory

Place your images in this folder following the structure below.

## Required Images

### Root Level (`assets/images/`)

1. **profile.jpg**
   - Your professional photo
   - Size: 400x400px (square)
   - Format: JPG or PNG
   - Tips: Good lighting, professional attire, neutral background

2. **favicon.png**
   - Website icon (appears in browser tab)
   - Size: 32x32px or 64x64px
   - Format: PNG with transparency
   - Simple, recognizable design

3. **og-image.jpg**
   - Social media preview image
   - Size: 1200x630px
   - Format: JPG
   - Include your name and title

4. **placeholder.jpg**
   - Fallback image (already created)
   - Used when images fail to load

## Project Images (`assets/images/projects/`)

Create this folder and add:
- `project1.jpg` - Screenshot of first project
- `project2.jpg` - Screenshot of second project
- `project3.jpg` - Screenshot of third project
- etc.

**Specifications:**
- Size: 800x600px or 1200x900px
- Format: JPG or PNG
- Show the app/project in action
- Clear, high-quality screenshots

## Blog Images (`assets/images/blog/`)

Create this folder and add:
- `blog1.jpg` - Featured image for first article
- `blog2.jpg` - Featured image for second article
- etc.

**Specifications:**
- Size: 600x400px or 1200x800px
- Format: JPG
- Relevant to article topic
- Professional quality

## Testimonial Images (`assets/images/testimonials/`)

Create this folder and add:
- `person1.jpg` - Photo of first testimonial author
- `person2.jpg` - Photo of second testimonial author
- etc.

**Specifications:**
- Size: 200x200px (square)
- Format: JPG or PNG
- Professional headshots
- Get permission before using

## Image Optimization

### Before Adding Images:

1. **Resize** to recommended dimensions
2. **Compress** using:
   - [TinyPNG](https://tinypng.com/)
   - [Squoosh](https://squoosh.app/)
   - [ImageOptim](https://imageoptim.com/) (Mac)
3. **Convert** to WebP for better performance (optional)
4. **Rename** with descriptive names (lowercase, hyphens)

### Tools:

**Free Online:**
- [TinyPNG](https://tinypng.com/) - Compression
- [Squoosh](https://squoosh.app/) - Resize & compress
- [Remove.bg](https://remove.bg/) - Background removal
- [Canva](https://canva.com/) - Create graphics

**Desktop:**
- GIMP (free)
- Photoshop
- Figma

## Naming Convention

Use descriptive, lowercase names with hyphens:

✅ Good:
- `android-music-player-app.jpg`
- `kotlin-compose-ui-library.jpg`
- `profile-photo-2024.jpg`

❌ Bad:
- `IMG_1234.jpg`
- `Screenshot 2024.png`
- `My Project.jpg`

## File Size Guidelines

Target file sizes:
- Profile photo: < 100KB
- Project screenshots: < 200KB
- Blog images: < 150KB
- Testimonial photos: < 50KB
- OG image: < 200KB

## Quick Start

If you don't have images yet:

1. **Profile Photo**: Use a professional photo from LinkedIn
2. **Projects**: Take screenshots of your apps
3. **Blog**: Use stock photos from Unsplash
4. **Favicon**: Create simple initials icon
5. **OG Image**: Create in Canva using template
